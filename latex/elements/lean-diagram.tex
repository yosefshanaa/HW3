\noindent\emph{הדגמת מעבר דו-כיווני} \term{(BiDi)}: לולאת
\en{Build--Measure--Learn} היא מנוע הלמידה של ה-\en{Lean Startup}~\cite{ries2011lean}
— \en{Ideas} $\rightarrow$ \en{Build} $\rightarrow$ \en{Measure} $\rightarrow$
\en{Learn}. השערה נבדקת נכתבת באנגלית
כשורה אחת:
\begin{quote}\en{\codefont If we add one-click checkout, weekly repeat purchases rise by 15\%.}\end{quote}
ואז חוזרים לעברית — רק כשהמספר מאומת מקדמים את הפיצ׳ר מניסוי אל המוצר.
